Software Portfolio

Word Processor (Java)
Git: https://github.com/ataylor89/Java/blob/main/WordProcessor.java
Description: A graphical word processor that has many color schemes, several look and feels, creates files, saves files, opens files, sends email, compiles source code, links object code, and runs program binaries.
Dependencies: javax.mail.jar
Compile: javac -classpath javax.mail.jar WordProcessor.java
Run: java -classpath javax.mail.jar:. WordProcessor

Magic Squares (Java)
Git: https://github.com/ataylor89/Java/blob/main/MagicSquares.java
Description: Magic Squares is a game that lets you put the numbers one through nine in the squares of a 9x9 grid. You win the game when every row, column, and subgrid has the numbers one through nine.
Resources: Sample puzzle
Compile: javac MagicSquares.java
Run: java MagicSquares

Prime Number Finder (NASM)
Git: https://github.com/ataylor89/nasm/blob/main/primes.asm
Description: Finds all prime numbers less than n.
Compile: nasm -fmacho64 primes.asm
Link: ld -macosx_version_min 10.7 -lSystem primes.o -o primes
Run: ./primes
